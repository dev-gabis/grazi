PROJETOS EM DESTAQUE

Contador de Clicks ultilizando Vue.js

Este projeto consiste em um contador de cliques desenvolvido utilizando o framework Vue.js. A estrutura básica do projeto inclui um arquivo HTML com a inclusão da biblioteca Vue.js, definindo um aplicativo Vue com um elemento principal identificado por "app". O aplicativo possui um estado de dados que inclui um contador iniciado em zero.


Clima atual em node

O projeto desenvolvido no Bootcamp da Generation Brasil no programa de Desenvolvimento Full-Stack consistiu na criação de uma aplicação web completa, com um front-end e um back-end. O foco do projeto era criar uma plataforma para conectar empresas e estudantes em busca de oportunidades de estágio. Para isso, foi utilizado um conjunto de tecnologias modernas e populares no mercado, como o framework Spring Boot para o back-end e o framework Angular para o front-end.


Projeto To-do list Nest.js

O projeto To-do list Nest.js é um projeto que permite ao usuário criar e gerenciar uma lista de tarefas. É desenvolvido usando o framework Nest.js, que é baseado no Node.js, o que permite a construção de aplicativos escaláveis e eficientes. O aplicativo tem uma arquitetura modular, onde cada recurso é definido em um módulo separado, tornando-o altamente organizado e fácil de manter. Ele também utiliza o TypeScript, que traz recursos adicionais de tipagem estática para melhorar a segurança e a produtividade do desenvolvedor.
